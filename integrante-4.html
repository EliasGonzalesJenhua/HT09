Operacion 02: configuracion del cluster

Este integrante explica como se prepara el cluster HA y que requisitos se necesitan para que las VM puedan recuperarse ante fallos.

Trabajo que va a leer

Explicacion

El cluster permite que las VM criticas se recuperen en otro nodo cuando un host falla. Para lograrlo, los hosts deben compartir red, almacenamiento y politicas HA.

La configuracion correcta del cluster convierte un fallo de host en una recuperacion controlada.

Resultados del cluster

Antes del fallo
  • Hosts operativos.
  • VM en Host 1.
  • Storage disponible.
Estado: normal
Durante failover
  • Se detecta caida.
  • HA elige Host 2.
  • VM se reinician.
Respuesta: automatica
Despues
  • Servicio continuo.
  • Interrupcion minima.
  • Host fallido se recupera.
